# H1 Kotiläksy

## Tiivistelmät

SSH public key - Login without password
- SSH-palvelin asennetaan sudo apt-get install ssh ja otetaan käyttöön komennolla sudo systemctl enable --now ssh.
- Toimivuuden voi testata ottamalla yhteyden omaan koneeseen
- Avainta luodaan komennolla ssh-keygen
- Päätteinen .pub-tiedosto on se, joka jaetaan palvelimille. Oma salainen avain pidetään vain itsellä.
- Komento ssh-copy-id localhost kopioi julkisen avaimen kohteen authorized_keys-tiedostoon, minkä jälkeen salasanaa ei enää kysytä.

Hello Ansible
- Ansible on konfiguraationhallintatyökalu. 
- Tiedostoon hosts.ini listataan hallittavat koneet.
- Playbook määrittää, mitä rooleja suoritetaan millekin koneille.
- Rooli sisältää tehtäviä, kuten tiedoston luominen /tmp/ hakemistoon.
- Ansible näyttää muutokset (changed=1), mikä auttaa ymmärtämään mitä tehtiin.
